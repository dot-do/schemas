# Product Documentation Knowledge Base

A comprehensive RAG (Retrieval-Augmented Generation) knowledge base containing all product documentation, updated daily and used by customer support agents.

## Overview

This knowledge base indexes:
- Product manuals and guides
- API documentation
- Troubleshooting articles
- FAQ content
- Video transcripts
- Release notes

## Architecture

### Indexing Strategy

**Hybrid approach** combining:
- **Semantic search**: Vector similarity using embeddings
- **Keyword search**: BM25 for exact matches
- **Reranking**: Cohere rerank-v3 for relevance

### Chunking Strategy

- **Chunk size**: 512 tokens with 128-token overlap
- **Metadata preserved**: Document title, section, category
- **Dynamic chunking**: Respects natural boundaries (headings, paragraphs)

### Update Process

1. Daily sync from documentation repository
2. Incremental updates for changed documents
3. Automatic re-embedding and re-indexing
4. Version tracking for rollback capability

## Usage

### Search Query

```typescript
const results = await knowledgeBase.search({
  query: "How do I reset my account password?",
  topK: 5,
  rerank: true,
  filters: {
    category: 'authentication',
    lastUpdated: { gte: '2025-01-01' }
  }
})
```
