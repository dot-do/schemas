# Deliverability Report Schema

Comprehensive report of email deliverability metrics, reputation, and recommendations.

## Report Components

### 1. Overall Score (0-100)

Weighted calculation:
- 40% Sender reputation
- 30% Delivery rate
- 20% Engagement (opens/clicks)
- 10% DNS authentication

### 2. Status Indicators

- **Excellent** (90+): All metrics optimal
- **Good** (70-89): Acceptable performance
- **Warning** (50-69): Issues need attention
- **Critical** (<50): Immediate action required

### 3. Key Metrics

**Deliverability:**
- Delivery rate > 98% (excellent)
- Bounce rate < 2% (excellent)
- Complaint rate < 0.05% (excellent)

**Engagement:**
- Open rate: 15-25% (good for cold email)
- Click rate: 2-5% (good)
- Reply rate: 1-3% (excellent)

### 4. Reputation Factors

**DNS Authentication:**
- SPF: Prevents spoofing
- DKIM: Email signing
- DMARC: Policy enforcement

**Sender Score:**
- 90-100: Excellent reputation
- 70-89: Good reputation
- 50-69: Questionable reputation
- <50: Poor reputation

**Blacklists:**
- 0 listings: Clean
- 1-2 listings: Warning
- 3+ listings: Critical

### 5. Issue Categories

**Critical Issues:**
- Bounce rate > 10%
- Complaint rate > 0.5%
- Found on high-authority blacklist
- Sender score < 50

**Warning Issues:**
- Bounce rate > 5%
- Complaint rate > 0.1%
- SPF/DKIM/DMARC not passing
- Sender score < 70

**Info Issues:**
- Low engagement rates
- Incomplete warmup
- Missing optional configurations

### 6. Recommendations

**Immediate Actions:**
- Fix DNS authentication
- Clean email list
- Request blacklist removal
- Pause sending (if critical)

**Short-term (1-2 weeks):**
- Improve content quality
- Adjust sending frequency
- Complete IP warmup
- Segment by engagement

**Long-term (1+ months):**
- Build sender reputation
- Optimize send times
- A/B test content
- Monitor competitors

## Report Frequency

- **Daily**: For domains under warmup or with issues
- **Weekly**: For active sending domains
- **Monthly**: For established domains with good reputation

## Trend Analysis

Compare current period to previous:
- Improving trends: ↑ in positive metrics, ↓ in negative
- Declining trends: ↓ in positive metrics, ↑ in negative
- Stable trends: <5% change

## Industry Benchmarks

Typical cold email benchmarks:
- Delivery rate: 95-98%
- Bounce rate: 1-3%
- Complaint rate: <0.1%
- Open rate: 15-25%
- Reply rate: 1-3%
