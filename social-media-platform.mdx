# Social Media Platform Connection Schema

## Overview

Platform connections manage OAuth credentials for publishing to social media platforms. Supports 6 platforms with token refresh and permission management.

## Data Model

```typescript
interface PlatformConnection {
  id: string
  userId: string
  platform: 'twitter' | 'linkedin' | 'facebook' | 'instagram' | 'tiktok' | 'youtube'
  status: 'active' | 'expired' | 'revoked' | 'error'

  // OAuth tokens (encrypted)
  accessToken: string
  refreshToken?: string
  expiresAt?: string

  // Platform identifiers
  platformUserId: string
  platformUsername: string
  displayName?: string
  profileUrl?: string
  avatarUrl?: string

  // Permissions
  permissions: string[]

  // Timestamps
  connectedAt: string
  lastUsedAt?: string
  createdAt: string
  updatedAt: string
}
```

## Supported Platforms

- **Twitter/X**: OAuth 2.0 with PKCE
- **LinkedIn**: OAuth 2.0
- **Facebook**: OAuth 2.0, Graph API
- **Instagram**: OAuth 2.0 via Facebook
- **TikTok**: OAuth 2.0
- **YouTube**: OAuth 2.0 via Google

## Service Implementation

-  **Service**: `workers/social-media-automation`
- **RPC**: `env.SOCIAL_SERVICE.connectPlatform()`
- **HTTP**: `POST /api/social/platforms/connect`
